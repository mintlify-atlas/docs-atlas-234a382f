# Weiroll

Weiroll is a flexible execution system designed to enable trust-minimized and gas-efficient scripting on Ethereum. This deployable implementation builds upon the original concept of weiroll with a simple self-contained deployment that can be used by any contract via delegatecall.

## What is Weiroll?

Weiroll provides a virtual machine (VM) that executes on-chain scripts composed of commands and state. It enables complex, multi-step operations to be executed atomically in a single transaction with minimal gas overhead.

This implementation wraps the audited [Enso Finance Weiroll VM](https://github.com/enso-finance/enso-weiroll) in a deployable contract that can be safely used by any protocol through delegatecall.

## Built on Audited Code

This project leverages the security-audited Weiroll VM implementation from Enso Finance, which itself is based on the original [weiroll](https://github.com/weiroll/weiroll) project. The DeployableVM wrapper adds an additional security layer by enforcing delegatecall-only execution.
